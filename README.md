# 📡 LichessRadar

> Never miss a prize tournament on Lichess.

LichessRadar is a free Chrome extension that lets you save any Lichess Arena or Swiss tournament in one click, get notified before it starts, and keep everything organized in one place.

---

## Features

- **One-click save** — click the Save to Radar button on any Lichess tournament page
- **Paste a link** — add any tournament directly from the popup
- **Notifications** — get alerted 30 min, 1h, 6h, 12h, or 24h before start (multi-alert)
- **Favorites** — star your most important tournaments
- **Personal notes** — add a quick note to any tournament
- **Prize filter** — manually mark tournaments with a prize and filter by them
- **Auto-sort** — sort by date, prize, favorites, or recently saved
- **Real-time sync** — popup updates instantly when you save from a page

### Manual (Developer Mode)
1. Download the latest release ZIP from [Releases](../../releases)
2. Unzip the file
3. Open Chrome and go to `chrome://extensions`
4. Enable **Developer mode** (top right)
5. Click **Load unpacked** and select the unzipped folder

## How It Works

1. Browse [Lichess](https://lichess.org) and find a tournament you like
2. Click the **📡 Save to Radar** button (bottom right of the page)
3. Open the extension popup to see your saved tournaments
4. Mark prize tournaments with ✅, add notes, star your favorites
5. Receive notifications before each tournament starts

## Privacy

LichessRadar stores all data **locally on your device** using `chrome.storage.local`.  
No data is ever sent to any external server.  
See [PRIVACY.md](PRIVACY.md) for full details.
